# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to short links

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- method-override

## Getting Started

- Install all dependencies (using 'npm install' command)
- Run the development web serving using the 'node express_server.js' command
